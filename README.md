# Enfield Pressure Washing

This folder now contains a reorganized React + Vite application for the Enfield Pressure Washing website.

## Structure

- `src/App.tsx` - main app shell with navigation, quote modal, chat widget, and footer
- `src/components/BeforeAfterSlider.tsx` - reusable before/after slider component
- `src/views/` - page components for Home, Services, Contact, and Testimonials
- `src/data.ts` - shared asset/fallback data and service/testimonial content
- `public/assets/` - migrated image and video assets

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Open the URL shown in the terminal.
